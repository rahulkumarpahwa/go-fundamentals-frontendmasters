What is the significance of the first letter's capitalization when defining struct fields in Go?
Capitalized first letters make struct fields exported and accessible from outside the package, while lowercase first letters keep fields private to the package

What are two common float types used for representing currency prices in Go?
float32 and float64, with float64 being more commonly used due to higher precision and default JSON encoding behavior

When importing packages in Go, what is imported at the package level?
The entire package is imported, not individual elements or names. All exported elements from the imported package become available


What are the key requirements for a Go file to be considered a valid executable program?
The file must have 'package main' declaration and contain a function named 'main'


How do you create a new Go module in a project?
Use the command 'go mod init' followed by a URL-based project name, which creates a go.mod file in the project root

What are the main HTTP verbs available in Go's standard library?
Get, Post, Put, and Delete are the main HTTP verbs available as functions in the net/http package

In Go, are HTTP methods like Get and Post actual methods?
No, they are functions in the http package, not methods. They look like methods but are actually exported functions

What does the strings.ToUpper() function do in Go?
It takes a string as input and returns a new string with all characters converted to uppercase, without modifying the original string

What is important to know about the len() function when used with strings in Go?
The len() function returns the number of bytes in a string, not the number of characters. This can be misleading with Unicode characters like emojis which may use multiple bytes

How can you create a dynamic URL in Go using string formatting?
Use fmt.Sprintf() with a URL template and pass variables as arguments to create a dynamic URL string

What does the len() function return when used on a string in Go?
Number of bytes in the string

In Go's http package, how can you make a generic HTTP request?
Using the Request function where you specify the verb as an argument

What is the recommended way to return a nil rate in Go?
Use a pointer to the rate, which can be set to nil. This is typically done by using the star (*) symbol when defining the return type and using the ampersand (&) to get a pointer to an instance.

What HTTP status code indicates a successful response?
200 is the status code that indicates everything is okay in an HTTP response.

How can you read the entire body of an HTTP response in Go?
Use the io.ReadAll() function, which reads a stream synchronously and returns a slice of bytes or an error.

What is the recommended way to create a custom error message in Go?
Use fmt.Errorf() from the format package to create a new error with a formatted string, which allows you to include dynamic values in the error message.

Why is an HTTP response body represented as an io.ReadCloser?
Because HTTP responses are received in chunks over TCP, allowing for streaming and progressive downloading of data, rather than receiving the entire response at once.

How do you create a pointer to a struct instance in Go?
Using the & operator before the struct variable

What happens when using the dot operator with a pointer to a struct in Go?
The dot operator automatically dereferences the pointer and allows direct access to struct methods and properties, making it easier to work with struct pointers without explicitly dereferencing them.


Why did the application trigger repeated firewall alerts when using go run?
Each time go run is executed, it builds a new binary in a temporary directory, which is considered a new application from the operating system's perspective, causing repeated firewall alerts.

What is the type of the rate variable when calling an API method?
The rate variable is a pointer to the struct, not the struct itself, but the dot syntax allows direct access to its properties and methods.

What initial step is needed after receiving JSON from a server?
Parse the JSON, as it is currently just a string and needs to be converted into a usable data structure.

What issue occurs when initially printing the rate?
The rate is initially printed as a memory address (pointer) instead of its actual value, requiring further processing to display the desired information.
